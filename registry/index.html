This is the registry for the four namespaces defined by the Activity Streams specifications.

For convenience this registry includes a short human-readable description of each item. However, these descriptions are not normative; consult the corresponding specification for the normative definition of each item

Verbs Registry of verb keywords
Object Types Registry of object type keywords
Activity Components Registry of activity components and their JSON property names
Object Components Registry of object components and their JSON property names
Specifications Index of specifications included in this registry
